# Agent Instructions

## Code style
- Python: PEP8, no unused imports
- JavaScript: vanilla only, no frameworks
- Keep functions small and single purpose

## Behaviour
- Always run existing code before making changes
- Do not install packages without telling me first
- If unsure, ask one question before proceeding

## Git
- Commit messages must be short and descriptive
- Always work on a new branch, never commit directly to main
